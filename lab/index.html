Conventions

Global

Dragon uses a Small, Medium, Large system (S, M, L). While not yet global, you can already call the three sizes on Columns and Buttons.

Typography

Dragon provides beautiful typography for those who would like to include it. To activate it simply add the ".typography" class to a container (e.g. html for global).

Element Styling

To trigger element styling simply add the ".dragon" class to elemnts like: dd

Notes

Rows

Rows are optional where there is only one row in the container.

Grid

No need for colM12 where elements are always on separate rows.
